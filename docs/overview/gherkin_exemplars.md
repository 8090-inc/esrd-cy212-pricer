# Gherkin Exemplars (Source PDF Notes)

These exemplars illustrate the preferred structure for rules:
- Rule Title
- Description (business intent)
- Gherkin steps (Given/When/Then)
- Code Path Citations (COBOL file + line evidence)

The PDF examples are stored in-repo:
- docs/policy/policies/system-explaination-cms-esrd.pdf
- docs/policy/policies/sample-10-rules-cms.pdf

Use them as style references when authoring new `.feature` files.
