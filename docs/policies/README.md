# Policy and Example Sources

This folder contains example and policy source files used for rule authoring and citation work.

Contents:
- `system-explaination-cms-esrd.pdf`: Generated system overview for ESRD PPS pricer (example, not official CMS policy).
- `sample-10-rules-cms.pdf`: Generated Gherkin-style rule exemplars (example, not official CMS policy).
- `addendum-cms-1732-f-esrd-pps-aki-wage-index-file.zip`: CMS-1732-F Addendum A wage index file (official CMS source).
- `addendum_a/`: Extracted Addendum A contents (Excel, crosswalk, notes).
