# Policy Version Index

This index captures effective dates, CMS transmittal IDs, and release references for policy sources used in rule citations.

Source of truth: `docs/policy/policy_version_index.csv`.

Guidance:
- Each policy source must include effective date and transmittal ID when available.
- When a citation is used in Gherkin, reference the policy_id from this index.
- If a policy update changes a rule, record a new effective_date row.
